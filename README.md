# 🚀 My Personal Portfolio Website

Welcome to my personal portfolio website!
This project showcases my skills, projects, certifications, and learning journey as a Computer Science student interested in building systems and exploring cloud technologies.

🌐 **Live Website:**
https://aayushpandey05.github.io/Personal_Portfolio/

---

# 👨‍💻 About This Project

I built this portfolio to create a central place where I can present my projects, technical skills, and certifications.

While building this website, my goal was not just to make a static page but to design an **interactive and engaging experience** using modern frontend techniques and animations.

It reflects my interest in **clean UI design, performance, and interactive web development**.

---

# ✨ Features

Some of the key features implemented in this portfolio:

* Interactive **hero section with text scramble animation**
* **Typing effect** for dynamic role display
* **Custom cursor and ripple click effects**
* **Particle background animation**
* **3D tilt project cards**
* Smooth **scroll reveal animations**
* **Expandable certifications section**
* Animated **skill progress bars**
* Fully **responsive design for mobile and desktop**

---

# 🛠 Technologies Used

This project was built using:

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **GitHub Pages** for deployment

I focused on keeping the stack simple while implementing advanced UI interactions using pure JavaScript and CSS.
